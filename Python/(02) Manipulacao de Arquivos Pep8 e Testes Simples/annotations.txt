# Aula 10 - O que são ambientes virtuais e como criar com venv

# Comando para instalar o venv:
   # - py -m venv venv => Cria o ambiente virtual

# Comando para ativar o venv:
   # - Windows: venv\Scripts\activate
   # - Windows Powershell: .\venv\Scripts\activate
   # - MacOS/Linux: source venv/bin/activate

# Comando para desativar o venv:
   # - deactivate
